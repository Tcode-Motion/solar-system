## 🌌 About

A **high-fidelity, interactive 3D simulation** of our solar system built using **Three.js**. Featuring realistic planet textures, accurate orbital paths, moons, and celestial mechanics — all running live in the browser.

> 🎯 *Explore the entire solar system — from Mercury to Neptune — right in your browser.*

---

## 🪐 Features

| 🌟 Feature | 📖 Description |
|:---|:---|
| 🎨 **Realistic Textures** | Custom textures for the Sun, all 8 planets, and major moons |
| 🖱️ **Interactive Controls** | Orbit around planets, zoom through space freely |
| ☀️ **Dynamic Lighting** | Central point-light source simulating real Sun illumination |
| 🌍 **Inner Planets** | Mercury, Venus, Earth (Luna), Mars (Phobos & Deimos) |
| 🪐 **Gas Giants** | Jupiter (Galilean moons) and Saturn (Titan, Enceladus + rings) |
| 🌀 **Ice Giants** | Uranus and Neptune with accurate tilt and color |
| 🛸 **Orbital Paths** | Visible elliptical orbit rings for all planets |

## 🚀 How to Run Locally

### Option 1 — Clone & Open
```bash
# 1. Clone the repository
git clone https://github.com/Tcode-Motion/solar-system.git

# 2. Navigate into the project
cd solar-system

# 3. Open index.html in your browser
open index.html
```

### Option 2 — Live Server (Recommended)
```bash
# If you have VS Code, install Live Server extension
# Right-click index.html → Open with Live Server
```

### Option 3 — View Live Demo
🔗 **[https://tcode-motion.github.io/solar-system//](https://tcode-motion.github.io/solar-system/)**
