# 🚀 3D Solar System with Three.js

A high-fidelity, interactive 3D simulation of our solar system built using **Three.js**. This project features realistic planet textures, orbital paths, and celestial mechanics.

## 🪐 Features
- **Realistic Textures:** Custom textures for the Sun, all 8 planets, and major moons.
- **Interactive Controls:** Orbit around planets and zoom through space.
- **Dynamic Lighting:** A central point-light source representing the Sun.
- **Celestial Accuracy:** Includes details for:
  * **Inner Planets:** Mercury, Venus, Earth (Luna), and Mars (Phobos, Deimos).
  * **Gas Giants:** Jupiter (Galilean moons) and Saturn (Titan, Enceladus).
  * **Ice Giants:** Uranus and Neptune.

## 🛠️ Built With
- [Three.js](https://threejs.org/) - 3D Engine
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) - Logic
- [HTML5/CSS3](https://developer.mozilla.org/en-US/docs/Web/HTML) - Layout and UI
